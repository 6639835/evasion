# Security Policy

## Supported versions

Only the latest version on the `main` branch is actively maintained.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email **security@evasion.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive an acknowledgement within 48 hours. We aim to release a fix within 14 days of confirmation.
